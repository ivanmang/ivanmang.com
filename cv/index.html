    Projects👨🏻‍💻

    Ghost Chess (with Theodo)

    October 2018

    Using a mechanical arm, an electro-magnet, and an Arduino, we were able to build a first prototype of an API-controlled chess board. The goal of this project is to build an iOS, Android and web app that will allow users to control the chess board. We use React-Native as the front-end, NodeJS for the server

    Event Organizing WebApp - Evena

    May 2018

    Evena is a personal event organiser aimed at connecting the student community. Unlike other Event organising platforms for business use, we focus on small groups and emphasis on decreasing the workload of the organisers. This web application is built with Java, JavaScript and using Spring MVC as the framework.

    PintOS

    January 2018

    We designed and implemented the core parts for a operating system framework for the 80x86 architecture. It supports kernel threads, loading and running user programs, and a file system. The project is written in C.

    WACC compiler

    October 2017

    We designed and built a compiler in java for the WACC language from scratch.

    ARM 11 project

    May 2017

    We build an ARM11 Architecture emulator and assembler, which is part of the C programming course. It is also used as a synchronous audio and visual demonstration on the Raspberry Pi. We build a game called "Simon Says" using Raspberry Pi.

    The Engineering Education Scheme Wales

    August 2015

    Design a new product with 3D printer that solves real industrial problems with team members and present it in The Big Bang Fair We designed a quadcopter which can monitor the partial discharge of power cables in order to detect an upcoming failure.
